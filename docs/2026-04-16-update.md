## 変更詳細

### #1: `/focus` コマンド追加（🟡 推奨）

Claude Code v2.1.110 で追加。フォーカスビューのトグル。
以前は Ctrl+O がフォーカスビューを担当していたが、v2.1.110 から Ctrl+O は詳細トランスクリプト表示専用となり、
フォーカスビューは `/focus` コマンドに移動。

**After (Slash Commands セクションに追加):**
```
- `/focus` — フォーカスビューをトグル（旧 Ctrl+O）
```

### #2: `/tui fullscreen` コマンド追加（🟡 推奨）

Claude Code v2.1.110 で追加。現在の会話でちらつきなしレンダリング（alt-screen）に切り替えるコマンド。
`CLAUDE_CODE_NO_FLICKER=1` が env に設定済みの場合はデフォルトで有効だが、
セッション中に切り替えたい場合に使用可能。

**After (Slash Commands セクションに追加):**
```
- `/tui fullscreen` — フルスクリーン（ちらつきなし）レンダリングに切り替え
```
