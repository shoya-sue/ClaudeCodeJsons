## AI エージェント使用ポリシー（Minimal）

このプロジェクトでは **読み取り専用モード** で AI エージェントを使用します。

### 許可される操作

- ソースコード・設定ファイルの読み取り
- `git status` / `git diff` / `git log` / `git show` / `git branch`
- テストの読み取り・解説
- ドキュメントの読み取り

### 禁止される操作

- ファイルの作成・編集・削除
- git commit / git push
- パッケージのインストール
- テストの実行・変更
- 環境変数・シークレットへのアクセス

## 優先ツール

コード探索・解説に集中してください。提案のみ行い、実行は人間が判断します。

## 注意事項

- 提案する際は必ず根拠と影響範囲を説明すること
- 不明な点は推測せず確認を求めること
